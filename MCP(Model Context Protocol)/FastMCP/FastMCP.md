# FastMCP

FastMCP는 [[MCP(Model Context Protocol)]] 을 다루기 위한 표준 프레임워크입니다. 2024년에 FastMCP 1.0은 공식 MCP [[Python]] SDK에 통합되었으며, 현재 FastMCP 2.0은 유지,관리되고 있는 최신 버전으로, MCP 생태계 전반을 위한 툴킷을 제공합니다.

MCP 프로토콜은 강력하지만 직접 구현하려면 서버 설정, 프로토콜 핸들러, 콘텐츠 타입, 오류 처리 등 반복적인 작업이 많습니다. FastMCP는 이런 복잡한 부분을 모두 처리해주므로 **개발자는 핵심 도구 개발에만 집중**할 수 있습니다. 함수에 데코레이터를 추가하는 것만으로 대부분의 기능을 구현할 수 있을 정도로 간단합니다.
